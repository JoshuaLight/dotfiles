# dotfiles
_My dotfiles for different applications._ 

## Default Paths
- `.alacritty.yml`: `~/.alacritty.yml`
- `.vimrc`: `~/.config/nvim/init.vim` (NeoVim)
- `coc-settings.json`: `~/.config/nvim/coc-settings.json` (NeoVim)
- `config.fish`: `~/.config/fish/config.fish`
- `.ideavimrc`: `~/.ideavimrc`.
